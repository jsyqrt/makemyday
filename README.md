## 📖 项目简介

Make My Day 是一个纯前端的 AI 日程规划助手。用户可以通过文字或语音输入告诉 AI 自己想做的事情，AI 会智能地将这些事情分解成具体可操作的事件，并根据"紧急重要四象限"给出优先级建议。

## ✨ 主要功能

- 🤖 **AI 智能解析**：输入你的计划，AI 自动分解成具体事件并给出行动建议
- 📊 **四象限管理**：基于紧急重要四象限理论，科学管理时间
- 🎤 **语音识别**：支持语音输入（需要浏览器支持）
- ✏️ **灵活编辑**：可以编辑、删除、拖拽排序事件
- 💾 **本地存储**：数据保存在浏览器本地，无需注册登录
- 🎨 **美观界面**：现代化的 UI 设计，流畅的用户体验

## 🚀 快速开始

```bash
npm install
```


## ⚙️ 配置说明

首次使用需要配置 LLM API：

1. **API Key**（必需）：
   - 默认使用 [SiliconFlow](https://siliconflow.cn) API
   - 也支持其他兼容 OpenAI 格式的 API

2. **Base URL**（可选）：
   - 默认：`https://api.siliconflow.cn/v1`
   - 可修改为其他 API 地址

3. **Model**（可选）：
   - 默认：`deepseek-ai/DeepSeek-V2.5`
   - 可修改为其他模型

4. **语音识别**（可选）：
   - **选项 1（推荐）**：配置 SiliconFlow 语音识别 API Key（与 LLM API Key 相同）
   - **选项 2**：留空使用浏览器内置的 Web Speech API（仅支持 Chrome/Edge）

## 📱 使用指南

### 1. 输入计划

在输入框中描述你的计划，例如：

```
我要写一份项目报告，下午开会讨论新功能，
晚上去健身房锻炼，周末陪家人去郊游
```

### 2. AI 智能解析

点击"让 AI 帮我规划"按钮，AI 会将你的计划分解成具体事件，并给出：
- 清晰的事件标题
- 优先级评估（四象限）
- 具体的行动建议

### 3. 管理事件

- **编辑**：点击编辑按钮修改事件信息
- **删除**：点击删除按钮移除事件
- **拖拽排序**：在列表视图中拖动事件调整顺序
- **切换视图**：在列表视图和四象限视图之间切换

### 4. 四象限分类

事件会根据紧急程度和重要程度自动分类：

- 🔥 **紧急且重要**：立即去做（如紧急项目、重要会议）
- ⚡ **紧急但不重要**：授权他人（如电话接听、部分邮件）
- ⭐ **不紧急但重要**：计划去做（如学习、锻炼、规划）
- 📌 **不紧急也不重要**：稍后再做（如娱乐、闲聊）

### 5. 语音输入

应用支持两种语音识别方式：

**方式 1：SiliconFlow API（推荐）**
- 配置 Speech API Key（与 LLM API Key 相同即可）
- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 识别准确度高，使用 FunAudioLLM/SenseVoiceSmall 模型
- 录音完成后自动转换为文字

**方式 2：浏览器内置（免费）**
- 留空 Speech API Key 即可使用
- 仅支持 Chrome、Edge 浏览器
- 完全免费，无需配置 API
- 实时语音转文字

## 🌟 特色亮点

1. **纯前端应用**：无需后端服务器，可部署到任何静态托管服务
2. **隐私保护**：数据存储在本地浏览器，不上传到服务器
3. **开源免费**：完全开源，可自行部署和定制
4. **AI 驱动**：利用大语言模型的能力，智能理解和规划
5. **响应式设计**：支持桌面和移动设备

## ⚠️ 注意事项

- 数据存储在浏览器本地，请勿清除缓存以避免数据丢失
- 推荐配置 SiliconFlow 语音识别 API，支持所有浏览器
- 如果不配置语音识别 API，仅 Chrome/Edge 浏览器支持语音输入
- 需要自行配置 LLM API Key，建议使用 SiliconFlow 等服务
